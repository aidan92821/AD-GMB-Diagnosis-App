# About the taxa classifier
The taxa classifier used in this project is from [the qiime2 classifier repository](https://library.qiime2.org/data-resources). Specifically it is [this silva 138 OTUs full length sequences classifier](https://data.qiime2.org/classifiers/sklearn-1.4.2/silva/silva-138-99-nb-classifier.qza).

If for some reason this gets removed from the site, try downloading a different silva 138 classifier into this directory and updating the `CLASSIFIER` constant in `pipeline.py`.
